Sudoku - Game Description

  • Sudoku is a game made up of nine 3x3 grids. The goal of the game is to fill a 9x9 board with these numbers in a way that there is a unique number in each row, column, and 3x3 configuration.

  • Often times, Sudoku boards have predetermined numbers, having the player find a working solution.

Preliminary Software Design

Model

Constants

SPEED_START: Value for the speed when timer is going. (1).
SPEED_END: Value for the speed when timer is stopped (0).
BLANK_VALUE: Represents the value for a blank square (-1).

Grid

  • The Grid object encapsulates an integer two dimensional array: int [9][9] grid.
  • Includes functions that correspond to checking the input number.
  • The BLANK_VALUE constant will first initialize all 81 positions.
  • Has properties for:
    1. Time Elapsed.
    2. Squares Left.
    3. Number of Errors.

Blocks

  • A block object represents one of the 9 3x3 blocks on the board.
  • Blocks are temporary objects for one-dimensional arrays for ease of checking a number's block uniqueness.
  • Example from image above: var tmpblock = [1,2,3,6,9,8,7,5,4];

Functions

Possible Functions that will come in useful:

checkProgress()
Returns true if checkInputtedNumber() succeeds so far for all inputted fields. Blank fields should be fine.

checkInputtedNumber()
Returns true if the functions checkRow(), checkColumn(), and checkBlock() all succeed along with input being a number 1-9.

checkRow()
Validate that the number is unique to the row.

checkColumn()
Validate that the number is unique to the column.

timer()
Track time, calls upon timerSpeed for starting and stopping.

timerSpeed()
Starts initially on puzzle generation. Stops on completion.
